Introduction

The Zelig team is pleased to announce the alpha release of Zelig version 4. This suite of R packages offers a user-interface identical to previous Zelig versions while simplifying the developer API and improving core functionality. For specifics concerning changes, see below or read the new Zelig Manual.

What's new in Zelig 4

  • Separation of zelig core from contributed models
  • An improved toolkit for rapid development of new Zelig models
  • Package construction wizard (zelig.skeleton)
  • zelig2 methods support an all-new, simplfied programming style
  • param methods support a sleeker, more formalized programming style
  • describe methods now run with less overhead
  • Brand new hook API, replacing previous zelig3 and zelig4 methods
  • Test coverage for all models
  • Integrated numerical and combinatorial math libraries
  • Roxygen-compliant documentation
  • Tools for publishing the complete Zelig documentation

How to Install

Any of the following methods will install Zelig 4 on your current system:
  • Type source("http://r.iq.harvard.edu/install_zelig.R") from within an R terminal
  • Follow the installation guide
  • Download the latest archived build and install, following the instructions within the Installation and Administration Guide
