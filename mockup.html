Zelig
Zelig is a single, easy-to-use program that can estimate, help interpret, and present the results of a large range of statistical methods. It literally is "everyone's statistical software" because Zelig uses (R) code from many researchers. We also hope it will become "everyone's statistical software" for applications, and we have designed it so that anyone can use it or add their methods to it. Zelig comes with detailed, self-contained documentation that minimizes startup costs for Zelig and R, automates graphics and summaries for all models, and, with only three simple commands required, generally makes the power of R accessible for all users. Zelig also works well for teaching, and is designed so that scholars can use the same program with students that they use for their research.

Zelig Add-on Packages

Zelig version 4 has added support for a modular programming style. This allows end-users to download Zelig add-ons that add functionality specifically catered to their programming needs, as well as provide contributors of Zelig to independently manage and statistical models without direct intervention from the Zelig core team.

For more information on contributing statistical packages to Zelig, please consult Zelig's Developers' Documentation or contact core developers via the Zelig mailing list.

A collection of Haravard-maintained Zelig packages is available on here at GitHub.

  • Bivariate Regressions (Git Repository)
    Adds support for bivariate logistic and probit regressions with added plotting functionality to appropriately describe regression information through contour plotting.
  • Generalized Additive Models (GAM) Regressions (Git Repository)
    Adds support for Generalized Additive Models (GAM) regressions as well as API support for describing smoothing functions. Specifically, this package contains GAM analogs of the common generalized linear models: logit, normal, poisson and probit.
  • Generalized Estimating Equations (GEE) Regressions (Git Repository)
    Adds support fpr Generalized Estimating Equations (GEE) regressions. Specifically, this package contains GEE analogs of the common generalized linear models: gamma, logit, normal, poisson and probit.
  • Mixed Effect Regressions (Git Repository)
    Add support for Mixed Effect regressions (also commonly referred to as Multilevel). Specifically, this package contains Mixed-effect analogs of the common generalized linear models: gamma, logit, ls, poisson and probit.
  • Multinomial Regressions (Git Repository)
    Adds support for two multinomial logit and probit regressions. No plotting support is currently available (due to the difficulty in visually representing data in 3 or more dimensions).
  • Ordinal Regressions (Git Repository)
    Adds support for ordinal logit and probit regressions. Additinally, this package contains useful plotting facilities for visually describing ordered quantities of interest of ordered choices.
  • Survey-Weighted Regressions (Git Repository)
    Adds support for survey-weighted regressions. Specifically, this package contains survey-weighted analogs of the common generalized linear models: gamma, logit, normal, poisson and probit.
